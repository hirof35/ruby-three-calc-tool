# 業務支援ツール (見積書・納品書・給与計算)

Ruby/Tk（Tcl/Tk）を使用した、軽量でシンプルなデスクトップ型業務支援アプリケーションです。
1つのウィンドウ内で、タブを切り替えて「見積書作成」「納品書作成」「給与計算」の3つの機能を実行できます。

## 🚀 機能概要

- **見積書作成タブ**: 宛名、品目、金額を入力し、消費税（10%）込みの計算書を自動生成します。
- **納品書作成タブ**: 納品先と納品内容を入力し、定型文付きの納品書テキストを生成します。
- **給与計算ツールタブ**: 時給と労働時間を入力し、基本給、所得税（概算5%）、雇用保険（概算0.6%）を差し引いた手取り金額（差引支給額）を計算します。
- **共通機能**: 出力金額はすべて自動で3桁カンマ区切り（例: ￥1,234,567）に整形されます。

## 🛠️ 動作環境・前提条件

- **Ruby**: 2.x 以降 / 3.x 対応
- **OS**: Windows, macOS, Linux (Tcl/Tk 環境が必要です)

### 依存ライブラリ
本アプリは `tk` 拡張ライブラリを使用しています。

## 📦 インストールと起動方法

### 1. 必要ライブラリのインストール

ターミナル（またはコマンドプロンプト）を開き、以下のコマンドを実行して `tk` gem をインストールします。

```bash
gem install tk
💡 注意 (macOS / Linux の場合)
システム環境によっては、OS側に Tcl/Tk のインストールが必要です。

macOS (Homebrewの場合): brew install tcl-tk

Ubuntu/Debian系: sudo apt-get install tcl-dev tk-dev

2. アプリケーションの起動
リポジトリをクローンするかスクリプトをダウンロードし、以下のコマンドで起動します。

Bash
ruby business_tool.rb
📝 使い方
アプリを起動するとウィンドウが表示されます。

上部のタブ（見積書作成 / 納品書作成 / 給与計算ツール）をクリックして機能を切り替えます。

各項目に必要な情報（テキストや数値）を入力します。

下部にある発行・計算ボタンをクリックすると、下段のテキストエリアに整形された結果が出力されます。

出力されたテキストは、ドラッグしてコピー＆ペーストが可能です。
